2.1.目标实例

非mvvm框架：
1.目标实例为一个dom对象（绑定在js实例对象内）
2.空间为普通的html dom对象（可静态创建）
3.射击事件由js实例对象监听
4.目标实例状态由js实例对象改变
5.射击事件成功改变目标组件状态，同时改变积分状态

mvvm框架：
1.目标实例为一个html组件
2.空间为一个html组件
3.射击事件绑定在目标组件上
4.目标组件状态（出现、消失）与某些状态值绑定
5.射击事件成功改变目标组件状态，同时改变积分状态
